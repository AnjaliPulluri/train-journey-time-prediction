# train-journey-time-prediction
Data Science project using Python and Machine Learning to predict train journey duration
# 🚆 Train Journey Time Prediction

## 📌 Overview
This project focuses on predicting train journey duration using machine learning techniques.  
The model is built using features such as distance covered and number of stops to estimate travel time.

---

## 🧠 Problem Statement
Train journey duration varies based on multiple factors.  
The goal of this project is to analyze the data and build a model that can predict journey time with reasonable accuracy.

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## ⚙️ Project Workflow
1. Data Loading and Understanding  
2. Data Cleaning (handling missing and invalid values)  
3. Feature Engineering (duration, stops, distance)  
4. Exploratory Data Analysis  
5. Visualization  
6. Model Building using Linear Regression  
7. Model Evaluation  

---

## 📊 Key Insights
- Journey duration increases with distance  
- Number of stops also impacts travel time  
- Relationship between distance and duration is mostly linear  

---

## 📈 Model Performance
The model was evaluated using standard metrics:
- Mean Absolute Error (MAE)  
- Root Mean Squared Error (RMSE)  

The Actual vs Predicted visualization shows that most predictions are close to actual values.

---

## 📷 Outputs
- Dataset preview  
- Visualization graphs  
- Distance vs Duration analysis  
- Final prediction graph  

---

## 🎯 Conclusion
The project successfully demonstrates how machine learning can be applied to real-world transportation data.  
Linear Regression provided a simple yet effective approach for predicting journey duration.
